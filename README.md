# Alfred-workflow-Jiumo

**Alfred workflow** for [Jiumo E-Book Search][Jiumo]


[鸠摩搜书][Jiumo]的**Alfred workflow**，可直接将搜索结果的链接复制到剪贴板，并打开浏览器进行下载。

## 安装

下载、安装 [Jiumo search.alfredworkflow](./Jiumo-search.alfredworkflow)

## 使用

```
jiumo 关键字
```
